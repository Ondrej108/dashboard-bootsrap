# Dashboard UI (Bootstrap)

A responsive dashboard interface built with Bootstrap, focused on creating a clean admin-style layout using reusable components.

🔗 Live Demo: https://ondrej108.github.io/dashboard-bootsrap

---

## Features

- Responsive layout (mobile, tablet, desktop)
- Admin-style UI structure
- Reusable components (cards, tables, buttons, badges)
- Sidebar navigation layout
- Consistent spacing using Bootstrap utilities

---

## Technologies

- HTML5
- CSS3
- Bootstrap (grid system, utilities, components)

---

## What I focused on

- Building responsive layouts with Bootstrap
- Structuring reusable UI components
- Combining Bootstrap utilities with custom CSS
- Maintaining clean and consistent design
